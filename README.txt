Этот проект создан для поиса людей по информации: номер телефона,
фотография человека, email и ФИО


НОМЕР ТЕЛЕФОНА:

По номеру можно узнать регион, оператора и страну. Возможно имя человека
 или его псевдонимы. Также при возможности социальные сети: Вк, Facebook, instagram

EMAIL:

При вводе email программа покажет пароль, если он есть в базе данных 
и возможные номера, имена и адреса привязанныые к этой почте

ПОИСК ПО ИМЕНИ:

Можно ввести как ФИО человека, так его псевдоним. В ответ программа выдаст
его номер телефона

ПОИСК ПО ФОТО:

Загружаете фото человекак, которогохотите найти, выбирайте его пол для более 
быстрого поиска (если пргорамма не нашла, можете попробовать сменить поле
выбора пола) после чего программа покажет все найденныые сходства человека,
а именно его имя, id "ВК" и ссылку на страничку. Поиск работает только
на территории Красноярска
